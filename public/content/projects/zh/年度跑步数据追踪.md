## 项目背景

一个使用 React 开发的跑步数据追踪和可视化应用。该应用允许用户记录跑步活动、查看统计数据，并通过交互式图表分析自己的跑步表现

## 功能特点

- 跑步数据可视化日历视图
- 统计面板显示关键指标
- 响应式设计，适配各种设备
- 跑步引言激励

## 技术实现

React 18
React Router v6
D3.js 用于数据可视化
HTML Canvas 用于图像导出
